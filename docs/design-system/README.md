# Design System

Owner: Engineering
Status: active
Last verified: 2026-05-08

Machine-readable extraction and governance contract for Wavefoundry UI surfaces.

The current design-system scope includes the local dashboard feature under `.wavefoundry/framework/dashboard/`. That dashboard uses a deliberately small shared contract:

- shell layout
- typography hierarchy
- spacing scale
- status-color semantics
- card and table primitives
- loading / empty / error state treatments

See [docs/architecture/design-system.md](/Users/coryhacking/Developer/wavefoundry/docs/architecture/design-system.md) for the architecture-level relationship between the dashboard assets and the design-system surface.
